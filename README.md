# AdvancedWidgetsApp
Advanced Widgets

We show in separated Activities the design and the use of some advanced android widgets as:
- CheckBox
- ProgressBar
- RatioButtons
- RatingBar
- SeekBar
- SwitchButton
and how to create our own button shapes.

Also, we create Vector Asset.
